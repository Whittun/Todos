# Todo App

Мини-приложение для просмотра списка задач из JSONPlaceholder.

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить dev-сервер:

```bash
npm run dev
```

Зайти на:

```
http://localhost:5173/
```

## Технологии

- React
- TypeScript
- Vite
- CSS Modules
- Feature-Sliced Design
- JSONPlaceholder API

## Реализовано

- загрузка первых 15 задач
- поиск по названию
- фильтр по статусу
- детальная модалка задачи
- загрузка пользователя для выбранной задачи
- состояния loading, empty и error
- адаптивная верстка
